# exercise-gene-expression-rnaseq
RNA-seq analysis of human exercise-induced gene expression changes using GEO dataset GSE120862. Pipeline includes FastQC, HISAT2 alignment, featureCounts, and DESeq2 differential expression analysis.

Exercise-Induced Gene Expression Analysis
Project Overview
This project investigates how aerobic exercise influences human gene expression using publicly available RNA-seq data from GEO accession GSE120862.
The goal was to identify genes and biological pathways associated with exercise-related adaptations that may support cognitive function and learning.

Dataset
  GEO: GSE120862
  Human skeletal muscle samples
  Baseline vs post-exercise comparison

Bioinformatics Pipeline
  FastQC quality control
  HISAT2 alignment to GRCh38
  featureCounts gene quantification
  DESeq2 differential expression analysis
  Gene Ontology interpretation

Tools Used
  Linux
  HPC Computing
  FastQC
  HISAT2
  featureCounts
  R
  DESeq2

Key Findings
  Significant transcriptional changes occurred following exercise.
PPARGC1A showed increased expression following exercise.
Pathway analysis indicated enrichment of mitochondrial and metabolic processes.
